Calculate UK Income Tax, National Insurance, and Employer NI contributions for the 2025-2026 tax year with an easy-to-use calculator.

== Description ==

The UK NI & Employer Tax Calculator is a comprehensive tool for calculating Income Tax (PAYE), National Insurance contributions, and total employer costs for the 2025-2026 tax year.

= Features =

* **Accurate Calculations**: Based on official HMRC rates and thresholds for 2025-2026
* **Complete Breakdown**: Shows Income Tax, Employee NI, Employer NI, and net income
* **Annual & Monthly Figures**: View both annual and monthly amounts
* **Percentage Display**: See what percentage of salary goes to each tax
* **Personal Allowance Support**: Enter your tax code for accurate calculations
* **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
* **Easy Integration**: Use simple shortcode `[uk_tax_calculator]` anywhere
* **Professional UI**: Clean, modern interface with smooth animations
* **No Configuration Needed**: Works out of the box

= Tax Year 2025-2026 Rates =

The calculator uses the following official rates:

**Income Tax (PAYE)**
* Personal Allowance: £12,570
* Basic Rate (20%): £12,571 to £50,270
* Higher Rate (40%): £50,271 to £125,140
* Additional Rate (45%): Over £125,140

**Employee National Insurance (Category A)**
* Primary Threshold: £12,570
* 8% on earnings between £12,570 and £50,270
* 2% on earnings above £50,270

**Employer National Insurance**
* Secondary Threshold: £5,000
* 15% on all earnings above £5,000

= Usage =

Simply add the shortcode to any page, post, or widget:

`[uk_tax_calculator]`

Users can then:
1. Enter their gross annual salary
2. Enter their tax code (optional, defaults to 1257L)
3. Click "Calculate Tax"
4. View the complete breakdown of taxes and net income

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to Plugins > Add New
3. Search for "UK NI Employer Tax Calculator"
4. Click "Install Now" and then "Activate"
5. Add the shortcode `[uk_tax_calculator]` to any page or post

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Go to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin
6. Add the shortcode `[uk_tax_calculator]` to any page or post

= Usage =

After activation, simply add the shortcode `[uk_tax_calculator]` to any page, post, or widget area where you want the calculator to appear.

== Frequently Asked Questions ==

= What tax year does this calculator cover? =

This calculator is specifically designed for the UK tax year 2025-2026 (6 April 2025 to 5 April 2026).

= Is the calculator accurate? =

Yes, the calculator uses official HMRC rates and thresholds for the 2025-2026 tax year. However, results are estimates and may vary based on individual circumstances. Always consult with HMRC or a qualified tax professional for official tax advice.

= Can I use this calculator for Scotland or Wales? =

The calculator currently uses England and Northern Ireland tax rates. Scotland has different income tax bands. Wales uses the same rates as England.

= What tax code should I use? =

The standard UK tax code for 2025-2026 is 1257L. If you have a different tax code, you can enter it in the calculator. The numeric part of your tax code represents your Personal Allowance.

= Does it calculate Student Loan repayments? =

No, this calculator focuses on Income Tax and National Insurance only. Student Loan repayments are not included.

= Can I customize the calculator appearance? =

Yes, you can add custom CSS to your theme to modify the calculator's appearance. The calculator uses class names prefixed with `uk-tax-calculator-`.

= Does it work with page builders? =

Yes, the calculator works with all major page builders including Elementor, Beaver Builder, Divi, and others. Just add the shortcode `[uk_tax_calculator]` in a shortcode widget or module.

= Is it mobile responsive? =

Yes, the calculator is fully responsive and works perfectly on all devices including smartphones and tablets.

= Does it require any configuration? =

No, the calculator works out of the box with no configuration needed. Simply install, activate, and add the shortcode.

= Can I use it on multiple pages? =

Yes, you can use the shortcode on as many pages as you want throughout your website.
